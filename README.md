## **專案初始化**

### 專案使用版本
- Ruby 2.7.3
- Rails 6.1.4
- PostgreSQL

### Install Project

1. Fork **Stork** Repository
2. bundle 安裝相關後端套件
3. yarn  安裝相關前端套件
4. Migration 資料庫遷移
5. foreman 啟動專案

```console
 $ bundle install
 $ yarn install
 $ rails db:migrate
 $ foreman start
```
